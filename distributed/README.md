# Распределённый режим

Распределённый режим — дополнительный функционал Cajeer Bots. Он не требуется для обычного local-запуска.

В local mode платформа запускает всех ботов сразу или каждого отдельно на одном сервере. Distributed mode добавляет Core Server / Control Plane и Runtime Agent / Worker Node для нескольких runtime-серверов.
