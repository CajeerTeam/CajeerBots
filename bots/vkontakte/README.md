# ВКонтакте

Standalone-обёртка адаптера.

```bash
python -m core run vkontakte
python bots/vkontakte/main.py
python -m bot.main  # если текущий каталог: bots/vkontakte
```


> Каноническая runtime-реализация адаптера находится в `core/adapters/`; этот каталог содержит standalone-обёртку и manifest для совместимости.
